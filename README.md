# Graphs
Лабораторные работы по графам

Задания:
# Базовая часть
Описать класс граф. Класс должен содержать следующие обязательные функции:
  1) конструктор класса – принимает путь к файлу, в котором хранится граф/орграф
  и тип (ориентированный/неориентированный);
  2) size – возвращает количество вершин в графе/орграфе;
  3) weight – принимает номера двух вершин, возвращает вес ребра/дуги,
  связывающего их;
  4) is_edge – принимает номера двух вершин и возвращает True, если в
  графе/орграфе есть соответствующее ребро/дуга, False, если ребра/дуги нет;
  5) add
  _
  vertex – принимает номер вершины графа, добавляет соответствующую
  вершину в граф;
  6) add_edge – принимает номера двух вершин и вес ребра, добавляет
  соответствующее ребро в граф;
  7) list_of_edges – без аргументов возвращает список всех рёбер графа; с
  аргументом (номер вершины) возвращает список рёбер графа, инцидентных
  данной вершине / дуг, исходящих из данной вершины.
  Реализовать программу для работы с классом, с интерфейсом в виде командной строки.
  Поддерживаемые команды: создать пустой граф, считать граф из файла, добавить
  вершину, добавить ребро, удалить вершину/ребро, вывести список всех вершин/ребер
  графа; проверить связность графа и вывести компоненты связности графа / компоненты
  слабой связности орграфа, найти кратчайший путь и его длину между двумя заданными
  вершинами, найти расстояния от вершины до всех остальных вершин, вывести
  минимальное остовное дерево графа (для орграфа – без учёта направлений дуг). <br><br>
  Варианты:<br>
  1. Хранение графа во входном файле:<br>
  a) файл со списком рёбер,<br>
  b) файл со списками смежности,<br>
  c) файл с матрицей смежности.<br>
  2. Метод хранения графа в классе:<br>
  a) в виде списков смежности,<br>
  b) в виде матрицы смежности.<br>
  3. Алгоритм нахождения расстояния между вершинами:<br>
  a) Дейкстры,<br>
  b) Беллмана-Форда;<br>
  4. Алгоритм нахождения остовного дерева:<br>
  a) Краскала;<br>
  b) Прима.<br>
